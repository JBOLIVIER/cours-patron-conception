# MapReduce

## Problème

Nous voulons **distribuer des traitements pour traiter des volumes de données importants**.

## Principe

Le principe proposé par MapReduce est le suivant :

* **Décomposer les données en lots (map)**
* **Traiter chaque lot**
* **Fusionner les résultats (reduce)**.
